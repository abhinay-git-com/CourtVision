# Scrum Plan

## Sprint Cadence

- Sprint length: 1 week
- Ceremony rhythm: lightweight, async-first
- Board type: Scrum
- Estimation: Fibonacci story points: 1, 2, 3, 5, 8

## Sprint 0: Setup and Strategy

**Dates:** 6 June 2026 - 11 June 2026

**Goal:** Establish team operating system, repository, baseline data plan, and modeling strategy.

Key outcomes:

- GitHub repository created
- Jira board imported
- Confluence home page created
- Initial dataset shortlist completed
- Modeling approach selected

## Sprint 1: Data Foundation

**Dates:** 12 June 2026 - 18 June 2026

**Goal:** Build clean player and match history datasets for men's and women's predictions.

Key outcomes:

- Public datasets selected
- Data ingestion scripts created
- Player identity normalization documented
- Grass-court and recent-form features drafted

## Sprint 2: Baseline Models

**Dates:** 19 June 2026 - 25 June 2026

**Goal:** Produce validated baseline rankings and match outcome models.

Key outcomes:

- Baseline ranking model
- Match winner model
- Backtesting report
- Explainability draft

## Sprint 3: Phase 1 Submission

**Dates:** 26 June 2026 - 28 June 2026

**Goal:** Finalize and submit top 8 predictions for men's and women's categories.

Key outcomes:

- Phase 1 prediction file
- Model explanation document
- Reproducibility checklist
- Submission confirmation

## Sprint 4: Tournament Monitoring and Revisions

**Dates:** 29 June 2026 - 6 July 2026

**Goal:** Monitor tournament progress and decide whether Phase 2 revisions improve expected score.

Key outcomes:

- Round-of-16 update process
- Revision decision memo
- Risk-adjusted prediction updates if needed

## Sprint 5: Knockout Predictions and Final Story

**Dates:** 7 July 2026 - 14 July 2026

**Goal:** Deliver knockout predictions and build the leadership presentation narrative.

Key outcomes:

- Quarterfinal, semifinal, and final predictions
- Set-difference estimates
- Final evaluation report
- Presentation outline

## Sprint 6: Final Presentation Readiness

**Dates:** 15 July 2026 - 24 July 2026

**Goal:** Prepare leadership-ready story, visuals, and speaking roles.

Key outcomes:

- Final deck
- Demo or model walkthrough
- Speaker notes
- Mock presentation feedback incorporated
