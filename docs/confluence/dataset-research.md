# Dataset Research

## Candidate Public Data Sources

- Jeff Sackmann tennis match datasets
- ATP and WTA rankings data
- Kaggle tennis datasets
- Official Wimbledon draw and match results once available
- Public player statistics sources

## Data Selection Criteria

- Public and allowed by hackathon rules
- Covers both men's and women's tennis
- Includes enough history for validation
- Can be refreshed during the tournament
- Easy to document and reproduce
