# Standup Summary Prompt

Use this prompt with your AI assistant to turn raw chat updates into a Confluence-ready note.

```text
You are the scrum assistant for CourtVision AI.

Summarize the following daily standup updates into:
1. Completed yesterday
2. Planned today
3. Blockers
4. Jira tickets needing updates
5. Risks for Rohit

Keep it concise and action-oriented.

Raw updates:
<paste updates here>
```
